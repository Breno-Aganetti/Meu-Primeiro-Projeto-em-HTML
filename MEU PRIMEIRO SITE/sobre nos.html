Quem somos

Meu nome é Breno Aganetti e esse é o meu primeiro projeto sem ajuda de professores.

mesmo quando eu eventualmente aprender sobre servidores, bancos de dados e sobre outras linguagens, eu não irei mecher nesse site, pois o objetivo dele é fazer o máximo que eu posso com o que sei no momento sem consultar ninguém e resolvendo os problemas na raça e na pesquisa.

além disso eu não vou mecher mais nesse site, por que a cada projeto eu vou me desafiar a fazer melhor, pois como se diz, antes feito que perfeito.

Ainda não tenho experiencia o suficiente para saber se o que eu fiz foi um bom trabalho ou uma gambiarra, mas este será meu primeiro registro de evolução, mesmo antes de aprender a usar o git e o git hub. Se você está lendo isso então eu já sei usar o git hub e nesse exato momento já estou fazendo outro projeto.

Me sinto incrivel, em menos de uma semana ja estou com uma tela cheio de comandos diferentes feitos por autoria própria, fazendo minhas próprias gambiarras kkkk e criando um projeto realmente funcional do zero. Segue abaixo uma "criação de conta" com os preenchimentos que aprendi até agora. Com excessão das imagens que peguei do curso, tudo isso inclusive as gambiarras eu fiz sozinho e pelo que vi ta bem diferente do site html do professor

Já vi muita treta entre programadores se html é ou não uma linguagem, na minha opinião é sim, pois se eu estou colocando comandos para a maquina fazer algo então pra mim é sim uma linguagem, joguei a merda e sai correndo kkkk quue comece a treta.

Nas observações do código dessa pagina eu vou colocar uma ou mais perguntas, vcês poderiam ler e/ou responder por favor?
